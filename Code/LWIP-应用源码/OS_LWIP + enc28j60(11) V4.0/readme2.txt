此LWIP 1.3.2 协议栈在火牛开发板上裸奔测试通过  之前用 版本1.4.0RC1 在调试telnet的时候发现有问题，后来更换1.4.0RC2版本（RC为测试版）解决了 但是后来再调试webservers的时候发现连接不上
调试了一天没结果，由于TCP UDP之前都实现了怀疑不应该程序问题 所以更换到 1.3.2版本，问题解决！

IP地址：  192.168.0.8
子网掩码：255.255.255.0
默认网关：192.168.0.1


1.UDP  UDP客户端 电脑IP 192.168.0.2 端口：1234 网卡灯不停的闪烁 为UDP不停的发送数据。此时电脑接收到的数据为123456789ABCDEFG(程序中要不段调用函数  etharp_tmr(); 否则程序执行一段后就不能发送数据了)

2.TFTP 网卡等不停闪烁期间 可以通过 windows命令窗口(桌面开始 ->运行)执行 telnet 192.168.0.8 回车后连接  然后按回车键会出现Hello. what is you name ?字样 

然后在里面随便敲几个字母按回车 会显示 Hello 加你刚才敲的字母 （注意：同时红色LED1会有相应的反应） 此时测试成功。
3  UDP服务器   服务器IP：192.168.0.2 端口 1025  远程电脑IP任意 	远程端口任意

4   tcp客户端  电脑IP为 192.168.0.2 端口：1026  本地端口 ：1026 
    连接后 将出现 This is LwIP TCP Client 在Luminary Cortex-M3上的测试！
